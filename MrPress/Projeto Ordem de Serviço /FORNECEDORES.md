## Ordem de prioridade por tipo de serviço

**Regra:** sempre enviar para a 1ª opção. Só passar para a próxima se a anterior não puder cumprir o prazo, e assim por diante até a última. A escolha é por disponibilidade/prazo, não por especialidade do trabalho.

Exemplo (adesivos): qualquer adesivo, inclusive corte especial, vai primeiro para a Imprime Sim. Se eles não derem conta no prazo, segue Grupo Selva, depois M2 e por último Rafael.

| Tipo de serviço | 1ª opção | 2ª opção | 3ª opção | 4ª opção |
| :--- | :--- | :--- | :--- | :--- |
| Adesivos | Imprime Sim | Grupo Selva | M2 | Rafael |
| Lonas e banners | Grupo Selva | M2 | Rafael | - |
| Placas (acrílico, PVC, PS), corte laser/router e adesivos especiais | Grupo Selva | - | - | - |
| Impressos gráficos gerais (cartão, flyer, calendário, crachá, envelope, papel timbrado, receituário, pasta, postal etc.) | Atual Card | Padrão Color | Print | - |
| Offset para trabalhos especiais (fora do catálogo Atual Card) | Carlos | Chico | - | - |
